ü§î The Challenge

Cryptocurrency and blockchain industries rely on a tool called a block explorer, which allows people to view all cryptocurrency transactions online. Examples include Etherscan, Ethplorer, and BlockExplorer. The main issue with these explorers is that they cater to advanced users, often neglecting design in favor of functionality.

‚ú® Our Approach

The goal was to create a custom explorer for Scroll's Data Transaction Token that would set a new standard for design and blockchain development, reaching beyond developers to a broader audience. SCRL token was later rebranded to Data Transaction Token, so you might find some discrepancies throughout this case study.

üôã My Role

As V.P., Creative Director, I was hands-on with the design of the interface and experience, working closely with the development team to ensure the website offered a seamless user experience. I collaborated with researchers and project managers to align the design within the development scope.

üìã Requirements

  • Contract address & Escrow account address
  • Live updated stats
  • Circulation movement published every 24 hours
  • Live Escrow tracker
  • List of transaction addresses and total number of holders
  • View transaction status & live transactions
  • Time in EST
  • Sort by token quantity, address, hash, and date
  • Volume graph
  • Visual indicators for transaction phases
  • Paging options

üé® The Design Process

I started with UI research by examining block explorers such as Etherscan, Dash, and Tron. Then, I turned to various platforms, including Pinterest, to find dashboard UI inspiration. To quickly create a mockup, I used a technique I call "Frankenstein-ing," where I pieced together elements from different interfaces to create a potential design. After approval from key players, I moved straight into a high-fidelity mockup due to time constraints.

üè™ One-Stop Shop

Users could visit the Data Transaction Token explorer to access all vital token information, laying the groundwork for interfaces in applications like Aster that rely on token purchases for data transactions.

üîé Transparent Information

Thanks to blockchain technology, every token interaction was recorded and published on the explorer, providing users with a clear understanding of the token ecosystem's current state.

üéâ Outcome

The Scroll community loved the modern and transparent platform, which paved the way for future enhancements like an Audit Explorer for legislation, finance, and regulatory bodies. Ledgers would display various token forks, user-tracked addresses, and supported main nets for Data Transaction Token as updates rolled out.
This project taught me a lot about working with a development team, striking compromises, and delivering the best possible user experience.
